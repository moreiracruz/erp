# Angular Frontend Generation

Follow `AGENTS.md` and `.github/skills/angular-frontend-generation.yaml`.

When working on frontend code:

1. Inspect existing Angular structure and reuse local component/service patterns.
2. Use TypeScript strict-friendly code, cohesive components, and reactive forms.
3. Keep HTTP access centralized in services/adapters.
4. Use guards and interceptors for authentication flows.
5. Do not rely on frontend-only checks for critical security or business rules.
6. Add or update focused component/service tests for changed behavior.
